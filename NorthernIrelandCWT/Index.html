While cleaning some data for the local stats dashboard, I noticed that recent South Eastern numbers were the same for Belfast and South Eastern HSCT prior to September 2023. In this script I’m going to explore the numbers and see what is/isn’t likely to be reliable.

Count

Belfast

Data cleaning 
belfastCount <- 
  left_join(mar24Count %>% 
              filter(hsc_trust == "Belfast") %>%
              rename(mar24_count = total_number_treated),
            dec23Count %>% 
              filter(hsc_trust == "Belfast") %>%
              rename(dec23_count = total_number_treated),
            by = c('date', 'hsc_trust')) %>%
  left_join(sep23Count %>% filter(hsc_trust == "Belfast") %>%
              rename(sep23_count = total_number_treated), 
            by = c('date', 'hsc_trust')) %>%
  left_join(sep22Count %>% filter(hsc_trust == "Belfast") %>%
              rename(sep22_count = total_number_treated), 
            by = c('date', 'hsc_trust')) 

belfastCount <- belfastCount %>%
  select(!hsc_trust) %>%
  pivot_longer(cols = !date,
               names_to = c("source", "other"),
               names_sep = "_",
               values_to = "count") %>%
  select(!other) %>%
  mutate(source = case_when(source == "sep22" ~ "September 2022 dataset",
                            source == "sep23" ~ "September 2023 dataset",
                            source == "dec23" ~ "December 2023 dataset",
                            source == "mar24" ~ "March 2024 dataset")) %>%
  mutate(source = factor(source, levels = c("September 2022 dataset", 
                                            "September 2023 dataset", 
                                            "December 2023 dataset", 
                                            "March 2024 dataset", 
                                            "June 2024 dataset")))
Chart code 
plot_ly(belfastCount) %>%

September 2022 and Septembeer 2023 data releases align, so I think they are likely reliable (you can click the legend to enable/disable certain lines).

Howeveer, I don’t think the South Eastern numbers are reliable in the December 2023, March 2024, or June 2024 datasets.

For the December 2023 dataset, the numbers are far too low, so I think this is just due to the Encompass switch.

The March 2024 datasets align with September 2022 and 2023 up until December 2023 when it takes a dive.

I think the June 2024 dataset is using Belfast numbers up until September 2024. Lets visualise to check (Belfast is not included in this data set at all).
