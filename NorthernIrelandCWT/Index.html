While cleaning some data for the local stats dashboard, I noticed that recent South Eastern numbers were the same for Belfast and South Eastern HSCT prior to September 2023, prompting a closer look at the dataset.

On this page I’m going to explore the numbers and see what is/isn’t likely to be reliable.

Belfast

Data cleaning 
belfastCount <- 
  left_join(mar24Count %>% 
              filter(hsc_trust == "Belfast") %>%
              rename(mar24_count = total_number_treated),
            dec23Count %>% 
              filter(hsc_trust == "Belfast") %>%
              rename(dec23_count = total_number_treated),
            by = c('date', 'hsc_trust')) %>%
  left_join(sep23Count %>% filter(hsc_trust == "Belfast") %>%
              rename(sep23_count = total_number_treated), 
            by = c('date', 'hsc_trust')) %>%
  left_join(sep22Count %>% filter(hsc_trust == "Belfast") %>%
              rename(sep22_count = total_number_treated), 
            by = c('date', 'hsc_trust')) 

belfastCount <- belfastCount %>%
  select(!hsc_trust) %>%
  pivot_longer(cols = !date,
               names_to = c("source", "other"),
               names_sep = "_",
               values_to = "count") %>%
  select(!other) %>%
  mutate(source = case_when(source == "sep22" ~ "September 2022 dataset",
                            source == "sep23" ~ "September 2023 dataset",
                            source == "dec23" ~ "December 2023 dataset",
                            source == "mar24" ~ "March 2024 dataset")) %>%
  mutate(source = factor(source, levels = c("September 2022 dataset", 
                                            "September 2023 dataset", 
                                            "December 2023 dataset", 
                                            "March 2024 dataset", 
                                            "June 2024 dataset")))

So for some reason the December dataset for Belfast is not reliable - the other 4 datasets all have the same numbers except for the December release. This isn’t a big problem though, since we have more recent data and the difference in December wasn’t huge.

September 2022 and September 2023 data releases align, so I think they are likely reliable (you can click the legend to enable/disable certain lines).

However, I don’t think the South Eastern numbers are reliable in the December 2023, March 2024, or June 2024 datasets.

For the December 2023 dataset, the numbers are far too low, likely due to the Encompass switch.

The March 2024 datasets align with September 2022 and 2023 up until December 2023 when it takes a dive.

I think the June 2024 dataset is using Belfast numbers up until September 2023. Lets visualise to check (Belfast is not included in this data release at all, so I am comparing to the March 2024 data).
