Chart function code
HSCT_Data_Chart <- function(df, areaName) {
  plot_ly(df) %>%
    add_trace(x = ~date, y = ~count, color = ~source,
              textposition = "none",
              hoverinfo = "text",
              text = ~ paste0("</br> <b>Quarter ending:", date, "</b>",
                              "</br> Source: ", source,
                              "</br> Total cases: ", format(count, big.mark = ",")),
              type = 'scatter', mode = 'lines') %>%
    layout(
      hovermode = "x",
      showlegend = TRUE,
      legend = list(orientation = 'h'),
      yaxis = list(
        title = list(text = "Number of patients starting\ntreatment each quarter", standoff = 15),
        range = c(0, plyr::round_any(max(df$count, na.rm = TRUE), 20, ceiling)),
        zeroline = F,
        showgrid = F,
        titlefont = list(family = "Poppins", color = "#000000", size = 18),
        tickfont = list(family = "Poppins", color= "#000000", size= 14)),
      xaxis = list(
        title = "", 
        zeroline = F,
        tickfont = list(family = "Poppins", color= "#000000", size= 14),
        titlefont = list(family = "Poppins", color = "#000000", size = 18)), #set these values to your x axis values
      title = list(text = paste0(areaName, " 62 DW count"), 
                   font = list(family = "Progress Medium", color = "#000000",size = 28),
                   x = 0.01),
      margin = list(t = 75) 
    ) %>%
    config(displaylogo = FALSE,
           toImageButtonOptions = list(format= 'png', # one of png, svg, jpeg, webp
                                       filename= paste0(areaName, '_Differences'),
                                       height= 500,
                                       width= 1200,
                                       scale= 1),
           modeBarButtonsToRemove = c('zoom', 'pan', 'select', 'zoomIn', 
                                      'zoomOut', 'autoScale', 'resetScale'))
}

While cleaning some data for the local stats dashboard, I noticed that recent South Eastern numbers were the same for Belfast and South Eastern HSCT prior to September 2023, prompting a closer look at the dataset.

On this page I’m going to explore the numbers and see what is/isn’t likely to be reliable.

Belfast

Belfast HSCT most recent reliable data

March 2024

Belfast moved to encompass in June 2024, and this has resulted in it being removed from the June 2024 data.

So for some reason the December dataset for Belfast is inconsistent - the other 3 datasets all have the same numbers except for the December release. This isn’t a big problem though, since we have more recent data and the difference in December wasn’t huge.

South Eastern was the first HSCT to switch to encompass in November 2023, with big inconsistencies seen since.

September 2022 and September 2023 data releases align, so I think they are likely reliable (you can click the legend to enable/disable certain lines).

However, I don’t think the South Eastern numbers are reliable in the December 2023, March 2024, or June 2024 datasets.

For the December 2023 dataset, the numbers are far too low, likely due to the encompass switch.

The March 2024 datasets align with September 2022 and 2023 up until December 2023 when it takes a dive.

I think the June 2024 dataset is using Belfast numbers up until September 2023. Lets visualise to check (Belfast is not included in this data release at all, so I am comparing to the March 2024 data).

The Northern Ireland Department of Health are currently not releasing national level data, instead releasing data covering 4 out of 5 of the HSCTs (with no Belfast data available in the latest June 2024 data). National level data is ultimately the sum of the HSCT data, so until all 5 HSCTs are reporting quality data again, it may be some time until we get a complete national picture again.
