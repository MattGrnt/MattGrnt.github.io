Trends over time template with ggplot

Show sample data 
CWT62 <- read_csv("C:/RProjects/cruk_ci_cross_pathway/CWT/Northern Ireland/SST Data/CWT_62day_By cancer site_2024-08-14.csv", show_col_types = FALSE) %>%
  clean_names() %>%
  filter(country == "England",
         ref_route == "Urgent Suspected Cancer") %>%
  mutate(date = as.yearmon(date))

Build the plot in ggplot first.

You can build the graph in plotly first, and the choice of which to use is just down to whichever you are more familiar with. We chose ggplot because we were using gghighlight which Iâ€™ll cover in a different section.

Show code 
plot <- ggplot(CWT62) +
  #Line 1
  geom_line(data = filter(CWT62, factor == "Breast"), 
            aes(x = date, y = percent_within_target),
            color = "#ff0087") +
  #Line 2
  geom_line(data = filter(CWT62, factor == "Lower GI"), 
            aes(x = date, y = percent_within_target),
            color = "#009cee") +
  scale_y_continuous(name = "y axis name",
                     limits = c(0, 1),
                     breaks = c(0, 0.25, 0.50, 0.75, 1),
                     label = c("0%", "25%", "50%", "75%", "100%")) +
  theme_minimal() 
plot

Pass the plot into ggplotly() which turns a ggplot graph into a plotly graph

Show code 
plot <- ggplotly(plot) %>%
  layout(
    yaxis = list(
      title = "Percent of patients (%)",
      range = c(0, 1),
      showgrid = T,
      automargin = T,
      tickformat = ".0%",
      titlefont = ciAxisFont,
      tickfont = ciTickFont),
    xaxis = list(
      title = " ",
      tickmode = 'auto',
      nticks = 10,
      showgrid = T,
      tickfont = ciTickFont,
      titlefont = ciAxisFont,
      #Set the dates on the axis to year
      tickformat = "%y"),
    #Set plot title
    title = list(text = "62 day standard: lung cancer performance",
                 font = ciTitleFont,
                 x = 0.01),
    #set margin for plot, right hand side is set to be wide to accomodate the annotations
    margin = list(t = 50, r = 275)) %>%
  config(displaylogo = FALSE,
         toImageButtonOptions = list(
           format = "png",
           filename = paste0("Graph"),
           width = 1400,
           height = 600)) 
  
plot

add on the annotations

Show code 
#Lung cancer 6m avg anno
plot %>%
  add_annotations(xref = 'x',
                  x = max(CWT62$date), # set to x axis data
                  # Use tail() to pick out the final data point of the trend line
                  # With this method you might have to filter or use the base (done with base R here)
                  y = tail(CWT62[CWT62$factor == "Breast",]$percent_within_target, 1), 
                  xanchor = 'left',
                  yanchor = 'middle',
                  # Set the text of the annotation
                  text = ~paste0(
                    format(
                      round(
                        tail(CWT62[CWT62$factor == "Breast",], 1)$percent_within_target 
                        *  100, 1), 
                      nsmall = 1),
                    '% - Breast cancer'),
                  font = list(family = "Poppins", color = "#ff0087", size = 16),
                  showarrow = TRUE,
                  # arrowhead symbol
                  arrowhead = 6,
                  xanchor = "right", yanchor = "centre", 
                  # position offsets for the annotation 
                  # useful for if text of different annotations overlaps
                  ay = 0, ax  = 15,
                  arrowcolor = "#ff0087", arrowwidth = 1, arrowsize = 1.2) #%>%

To output to image, you have two options.

  1. Use the camera icon on the plotly graph. The downside to this method is sometimes it the text comes out all weird

  2. Use orca to output as an image. You need to install orca first for this to be possible, but once it is working it is more reliable.

plot %>%  orca("Example graph", width = 1400, height = 600)
